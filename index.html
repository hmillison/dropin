Drop In CSS

A no thought CSS framework

Often when I'm working on personal projects, I want to have some nice styling other than the default stylesheet. For my purposes something like Bootstrap is too bloated and requires you to give everything tons of classes. I was inspired by Gloss CSS to make my own CSS framework that provides aestheically pleasing styles for the default stylesheet. Additionally, it provides a basic grid system using Flexbox. This allows you to... "drop in" the stylesheet and have a pretty webpage in seconds!!!
